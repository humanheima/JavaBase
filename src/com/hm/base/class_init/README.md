这个目录下，研究类的初始化过程

对于一个类而言，按照如下顺序执行：

* 执行静态代码块

* 执行构造代码块
* 执行构造函数
* 对于静态变量、静态初始化块、变量、初始化块、构造器，它们的初始化顺序依次是（静态变量、静态初始化块）>（变量、初始化块）>构造器。
